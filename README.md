# Cyber Vixen Studios

Websites, mobile apps, SEO, advertising, cloud solutions, and Vixen Care for businesses that want to build, manage, and grow their online presence.

## Development

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

## Brand

Site metadata, favicon, and browser tab branding use the Cyber Vixen Studios name and fox logo.
